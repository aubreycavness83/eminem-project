# SLIM SHADY - Eminem Fan Page

An interactive fan site celebrating Marshall Mathers (Eminem), one of the greatest rappers of all time. Features a curated collection of music videos, albums, and an interactive experience.

## Features

- **Hero Section**: Striking introduction with animated title
- **About Section**: Biography with key statistics (Grammy awards, records sold, albums)
- **Albums Gallery**: Showcase of Eminem's top studio albums
- **Favorite Songs**: Interactive music video player for 5 iconic tracks
  - Rap God
  - Lose Yourself
  - Godzilla
  - Without Me
  - Houdini
- **Interactive Music Lab**: Experimental beat-making and production interface
- **Stage Show**: Full-screen video experience with overlay effects
- **Responsive Design**: Optimized for desktop and mobile (320px+)

## Tech Stack

- **HTML5** with semantic structure
- **CSS3** with BEM naming and responsive breakpoints
- **Vanilla JavaScript** with IIFE pattern
- **Bootstrap 5** for baseline framework
- **HTML5 Video** for native playback
